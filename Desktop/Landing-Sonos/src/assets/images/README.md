# Imágenes para SoundZone Landing Page

Este directorio contiene las imágenes que se utilizan en el landing page de SoundZone. A continuación se describe cada imagen que necesitas agregar:

## Lista de Imágenes Requeridas:

### 1. raspberry-pi.jpg
**Descripción:** Una Raspberry Pi 4 moderna con pines GPIO visibles, colocada sobre un fondo blanco limpio. La placa debe ser claramente visible con su PCB verde característico, puertos USB, puerto Ethernet y ranura para microSD.
**Uso:** Sección "Introducción"
**Estilo:** Fotografía de producto profesional con buena iluminación y enfoque nítido.

### 2. audio-control.jpg
**Descripción:** Una interfaz de control de audio multizona moderna, mostrando múltiples zonas de audio, controles de volumen, y una interfaz intuitiva para gestionar el sonido en diferentes áreas.
**Uso:** Sección "Objetivos del Proyecto"
**Estilo:** Captura de pantalla o mockup de interfaz de usuario.

### 3. system-architecture.jpg
**Descripción:** Un diagrama de arquitectura del sistema que muestre la conexión entre el frontend React, backend Spring Boot, extensiones de navegador, y nodos Raspberry Pi distribuidos.
**Uso:** Sección "Componentes del Sistema"
**Estilo:** Diagrama técnico limpio con iconos y conexiones claras.

### 4. event-venue.jpg
**Descripción:** Una casa de eventos moderna con sistema de audio instalado, mostrando diferentes zonas de audio, altavoces, y el ambiente de un evento en desarrollo.
**Uso:** Sección "Casos de Negocio"
**Estilo:** Fotografía de ambiente profesional.

### 5. financial-chart.jpg
**Descripción:** Un gráfico financiero que muestre la inversión inicial, ahorros comparativos, y proyección de costos operativos del sistema SoundZone.
**Uso:** Sección "Proyección Financiera"
**Estilo:** Gráfico profesional con datos claros y colores corporativos.

### 6. implementation.jpg
**Descripción:** Imagen que muestre el proceso de implementación, instalación de Raspberry Pi, pruebas del sistema, y documentación técnica.
**Uso:** Sección "Recomendaciones"
**Estilo:** Fotografía de proceso técnico o collage de implementación.

### 7. success-story.jpg
**Descripción:** Una imagen que represente el éxito del proyecto, mostrando el sistema funcionando en una casa de eventos real, con clientes satisfechos o el equipo de desarrollo.
**Uso:** Sección "Conclusiones"
**Estilo:** Fotografía de éxito o testimonial visual.

## Especificaciones Técnicas:
- **Formato:** JPG o PNG
- **Resolución:** Mínimo 800x600px, recomendado 1200x800px
- **Tamaño:** Máximo 2MB por imagen
- **Calidad:** Alta resolución para web

## Notas:
- Todas las imágenes deben ser optimizadas para web
- Considera usar herramientas como TinyPNG para comprimir las imágenes
- Asegúrate de que las imágenes tengan licencias apropiadas para uso comercial
